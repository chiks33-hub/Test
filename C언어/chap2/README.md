# 마크다운(Markdown) 종합 가이드

## 1. 제목 (Headings)
# 제목 1 (가장 큼)
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
###### 제목 6 (가장 작음)

## 2. 텍스트 강조 (Emphasis)
*이탤릭체* 또는 _이탤릭체_
**볼드체(굵게)** 또는 __볼드체(굵게)__
***굵은 이탤릭체***
~~취소선~~
