# 마크다운(Markdown) 문법 정리

## 1. 제목(Heading)

```markdown
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
###### 제목 6
```

결과:

# 제목 1
## 제목 2
### 제목 3

---

## 2. 문단 및 줄바꿈

```markdown
첫 번째 문장

두 번째 문장

줄바꿈을 하려면 문장 끝에 공백 2개 입력  
다음 줄
```

---

## 3. 강조(굵게, 기울임)

```markdown
*기울임체*
_기울임체_

**굵게**
__굵게__

***굵게 + 기울임***
```

결과:

*기울임체*  
**굵게**  
***굵게 + 기울임***

---

## 4. 취소선

```markdown
~~취소선~~
```

결과:

~~취소선~~

---

## 5. 인용문(Blockquote)

```markdown
> 인용문
>> 중첩 인용문
>>> 3단계 인용문
```

결과:

> 인용문
>> 중첩 인용문
>>> 3단계 인용문

---

## 6. 순서 없는 목록

```markdown
- 항목 1
- 항목 2
  - 하위 항목

* 항목 1
* 항목 2

+ 항목 1
+ 항목 2
```

결과:

- 항목 1
- 항목 2
  - 하위 항목

---

## 7. 순서 있는 목록

```markdown
1. 첫 번째
2. 두 번째
3. 세 번째
```

결과:

1. 첫 번째
2. 두 번째
3. 세 번째

---

## 8. 체크박스

```markdown
- [x] 완료
- [ ] 미완료
```

결과:

- [x] 완료
- [ ] 미완료

---

## 9. 코드

### 인라인 코드

```markdown
`printf("Hello");`
```

결과:

`printf("Hello");`

### 코드 블록

````markdown
```c
#include <stdio.h>

int main() {
    printf("Hello World");
    return 0;
}
```
